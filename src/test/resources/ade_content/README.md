# Projet ADE pour tests automatiques d'intégration

## Les ressources
Les ressources doivent être créés conformément au descriptif du fichier [src/test/resource/ade_content/ressources.txt](ressources.txt).

Les dossiers sont précédés du signe '*' et les feuilles du signe '-'.

La notation {email} indique que l'adresse mail de la ressource doit être précisée (connue dans LDAP et configurée dans le paramétrage du script de tests automatiques)

## Les activités
Les activités doivent être créés conformément au descriptif du fichier [src/test/resource/ade_content/activites.txt](activites.txt).

Les dossiers sont précédés du signe '*' et les feuilles du signe '-'.

## Les affectations et placements
Les affectations (activités/ressources) et les placements doivent être créés conformément au descriptif du fichier [src/test/resource/ade_content/affectations_et_placements.csv](affectations_et_placements.csv).
